# Arcaea Story Wiki Data

This repository contains the Lua data files for Chinese Arcaea Wiki. The data is sourced from the Arcaea game APK and is updated regularly to reflect changes in the game.

The outputs should be synchronized to [Module:Story/data](https://wiki.arcaea.cn/Module:Story/data) (and [/zh-hans](https://wiki.arcaea.cn/Module:Story/data/zh-hans), [/zh-hant](https://wiki.arcaea.cn/Module:Story/data/zh-hant), [/en](https://wiki.arcaea.cn/Module:Story/data/en)) after each update.
